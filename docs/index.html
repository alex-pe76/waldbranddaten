Datenschutzerklärung

Zuletzt aktualisiert: April 2026

Diese Datenschutzerklärung gilt für die iOS-App Waldbrandstufe (nachfolgend „App"), entwickelt von Alexander Petring.

Welche Daten werden verarbeitet?

  • Standortdaten: Die App fragt deinen aktuellen Standort ab, um die nächstgelegene DWD-Wetterstation zu ermitteln und dir die lokale Waldbrandgefahrenstufe anzuzeigen. Der Standort wird ausschließlich lokal auf deinem Gerät verarbeitet und nicht an Server übertragen oder gespeichert.
  • Waldbranddaten: Die App ruft öffentlich zugängliche Waldbrandgefahrendaten vom Deutschen Wetterdienst (DWD) ab. Dabei werden keine personenbezogenen Daten an den DWD übermittelt.

Weitergabe an Dritte

Es werden keine personenbezogenen Daten an Dritte weitergegeben, verkauft oder für Werbezwecke genutzt.

Analyse & Tracking

Die App enthält keine Analyse-Tools, keine Werbenetzwerke und kein Tracking.

Berechtigungen

  • Standort (während der Nutzung): Wird ausschließlich zur Bestimmung der nächsten Wetterstation verwendet. Du kannst die App auch ohne Standortfreigabe nutzen, indem du einen Ort manuell auswählst.

Datensicherheit

Da keine Daten erhoben oder übertragen werden, entfällt eine serverseitige Datenspeicherung vollständig. Alle Informationen verbleiben auf deinem Gerät.
